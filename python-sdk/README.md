# Jupiter RFQ Market Maker Python SDK

Python SDK for Jupiter RFQ market maker integration via gRPC streaming.

## Installation

```bash
cd python-sdk
python venv
pip install -e .
```

Run the example:
```bash
python examples/production_streaming.py
```

## Requirements

- Python 3.8+
- asyncio
- grpcio, protobuf, solders, base58
